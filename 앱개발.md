### 최근 수정 및 반영 사항 (v1.14.15)
- **구간 반복(Repeat Loop) 기능 추가 및 UI 개선**:
  - Output FX 트랙 영역 드래그로 반복 구간 설정, 시작/끝 라인 조절 핸들, 삭제 `[X]` 버튼 구현.
  - Repeat 활성화 시 재생 중 루프 안으로의 스냅 및 구간 내 반복 재생 지원.
  - 반복 구간 표시를 전체 트랙을 세로로 가로지르도록 연장하되 Output FX 트랙 하단 경계선에 정렬 및 짙은 amber 톤 시인성 보완.
- **Electron 네이티브 오디오 엔진 브릿징**:
  - `audio-bridge.js` 내 `setLoopRange(range)`, `setRepeatPlayEnabled(on)` 추가 및 C++ 엔진 연동.
  - 재생 시작 시 스냅, 정지 시 시작점 이동, 플레이헤드 wrapping 계산 반영.
  - 30ms 주기적 감시(`startLocalTickLoop`) 내 루프 감지 시 seek 명령 전달 및 seek 후 500ms 동안 수신되는 stale position 프레임 무시(`lastSeekSentAt` 타임스탬프)로 화면 떨림 해결.
- **Time Ruler & 타임라인 미니맵 연동**:
  - 오버레이 `zIndex`를 `9`로 올려 Time Ruler 영역까지 반복 구간 하이라이트 확장.
  - Time Ruler 상단 좌우 끝에 `clipPath` 기반의 sticky 삼각형 마크 추가.
  - playhead 컴포넌트들의 `zIndex`를 `10`으로 올려 레이아웃 정렬.
  - 타임라인 미니맵(`TimelineMinimap`)에 반복 구간 및 활성 상태 반영 음영 렌더링.

## 2. 릴리즈 전 남은 작업

1. **최종 배포 빌드 생성**
   - Windows 설치본/포터블 빌드가 필요하면 `npm run dist:win` 실행.
   - 네이티브 엔진 포함 여부, `bin/FocusDAW-AudioEngine.exe`, Help 이미지, ffmpeg 경로가 패키징 결과에 포함되는지 확인.

2. **최종 스모크 테스트**
   - 앱 실행/종료, 프로젝트 신규 생성, 오디오 파일 import, 최근 프로젝트 복원.
   - 재생/정지/seek/repeat, Solo/Mute, 트랙 볼륨/팬, Mixer/Advanced Pan/Advanced Effects 창 열기.
   - Vari BPM, Vari Key, Fade in/out, Volume Automation이 재생과 Export에서 모두 반영되는지 확인.
   - MP3/WAV Export 완료 및 결과 파일 재생 확인.

3. **시험 문서 확인**
   - 남은 수동 시험 항목이 있으면 [시험.md](시험.md)에만 유지.
   - 완료된 시험은 [시험-아카이브.md](시험-아카이브.md)로 이동.

## 3. 향후 개발 후보

### 우선순위 높음

1. **클립 편집 UI 재개 여부 결정**
   - 엔진의 `splitClip`/`joinClips`와 관련 코드(`ToolBar`, `handleSplit/Join`, 단축키)는 남아 있으나 화면에서는 숨김 상태.
   - 재개 시 가위/잇기/선택 툴바, 클립 인스펙터, 클립 이동/재정렬 UI를 함께 설계.

2. **성능 개선 Phase 6 고도화**
   - 대용량/다트랙 프로젝트에서 peak 캐시, Web Worker 디코딩, automation 곡선 memoize, 스크롤/줌 성능 추가 개선.
   - 장시간 세션의 메모리 사용량과 Export 캐시 무효화 정책 점검.

### 중간 우선순위

3. **트랙 재정렬 / 클립 타임라인 이동**
   - 트랙 헤더 드래그 순서 변경.
   - 선택 툴 기반 클립 드래그 이동 및 정렬 보조 기능 검토.

4. **자동 업데이트 + CI 릴리즈 파이프라인**
   - `electron-updater` + GitHub Releases 연동.
   - `electron-builder --publish` 및 릴리즈 아티팩트 자동 생성 검토.

5. **리버브 IR 개선**
   - 절차적 노이즈 IR을 실측 IR 파일 로딩/프리셋 방식으로 확장.
   - Ambience 프리셋과 네이티브/웹 fallback 경로의 사운드 일치성 확인.

### 검토 항목

6. **EQ 밴드 표현 통일**
   - OUTPUT FX 트랙의 3밴드(Low/Mid/High)와 Mixer Master/Advanced EQ의 9밴드 표현 차이를 유지할지, 조작 경험을 통일할지 검토.

7. **VST3/고급 미터링 장기 검토**
   - VST3 플러그인 로딩, True Peak Meter, 고급 LUFS Meter, 외부 플러그인 체인 지원 가능성 조사.

8. **Cloud Sync 장기 검토**
   - `.focus` 프로젝트 파일과 원본 Stem 파일을 Google Drive 등 클라우드 저장소와 동기화하는 방식 검토.

## 4. 보류 항목

- **클립 편집 UI(1-D)**: 현재 미사용 방침. 엔진 코드는 유지하되 화면 노출은 보류.
- **변동 BPM / Tempo Map 분석**: 현재 전체 곡 비례 스케일링 구조에서는 필수 아님. 비트 그리드/워프 편집을 도입할 때 재검토.
- **AudioWorklet 기반 실시간 DSP 재시도**: 현재 네이티브 JUCE 엔진과 WebAudio fallback이 동작하므로 즉시 착수하지 않음.
