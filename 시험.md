## 1. 시험 항목

### B-200 — 구간 반복(Repeat) ON 시 플레이바 구간 진입 (v1.15.1)

**[시험 환경 준비]**
- `npm start`로 앱 실행(네이티브 엔진 연결 상태에서 시험 — 본 버그는 네이티브 전용).
- 오디오 파일 import(또는 데모 로드)하여 트랙이 있고 재생 가능한 상태.

**[시험 항목 및 방법/절차]**
1. **재생 중 + 구간이 플레이바 뒤쪽(이후 시점)** — (핵심 회귀)
   - 재생을 시작한다(플레이바가 예: 3초 부근을 지나는 중).
   - Output FX 트랙 영역에서 플레이바보다 **뒤쪽(예: 10~20초)** 에 반복 구간을 드래그로 만든다.
   - `Repeat` 버튼을 ON 한다.
   - 기대: 플레이바가 **즉시 10초(구간 시작점)로 이동**해 10~20초 구간을 반복 재생한다. (구간만 그려지고 3초에서 선형 재생되지 않아야 함.)
2. **재생 중 + 구간이 플레이바 앞쪽(이전 시점)**
   - 재생 중 플레이바보다 앞쪽(이미 지난 구간)에 반복 구간을 만들고 Repeat ON.
   - 기대: 플레이바가 구간 시작점으로 이동해 반복(기존과 동일하게 정상).
3. **정지 상태 + 구간 반복 ON** — (스냅 추가)
   - 정지 상태에서 플레이바 밖에 반복 구간을 만들고 Repeat ON.
   - 기대: 플레이바가 즉시 구간 시작점으로 이동한다. 이후 Play 시 구간부터 반복.
4. **구간 안에서 ON**
   - 플레이바가 이미 구간 안에 있을 때 Repeat ON.
   - 기대: 플레이바가 튀지 않고 현재 위치를 유지하며 구간을 반복.
5. **구간 끝 도달 시 wrap**
   - 위 상태에서 재생을 이어가 구간 끝에 도달하면 구간 시작점으로 되돌아가 반복되는지 확인.

**[통과 기준]** 1~5 기대 동작이 모두 충족되고, 오디오와 화면 플레이바가 어긋나지 않으며 콘솔 오류가 없다.

> ※ 참고: 재생 중 Repeat 구간을 **뒤쪽으로 이동/드래그** 시 오디오가 즉시 따라가게 하는 시도(v1.15.2)는 seek 폭주로 '징~~~' 잡음이 발생하여 원복함(미해결 과제).

### B-199 — 타임라인 zoom/스크롤 마우스 휠 단축키 (v1.15.0)

**[시험 환경 준비]**
- `npm start`로 앱 실행.
- 오디오 파일 2개 이상 import(또는 데모 로드)하여 타임라인에 트랙과 파형이 보이게 함.
- 타임라인이 화면보다 길도록 TIME zoom을 어느 정도 확대해 둠(가로 스크롤이 가능한 상태).

**[시험 항목 및 방법/절차]**
1. **Ctrl+휠 — 커서 중심 zoom in**
   - 타임라인 중앙의 특정 파형 지점(예: 어떤 클립의 시작 지점)에 마우스 커서를 올린다.
   - `Ctrl`을 누른 채 휠을 **위로** 굴린다.
   - 기대: 타임라인이 확대되고, **커서 아래에 있던 지점이 화면상 거의 같은 X 위치에 그대로 머문다**(좌측 기준으로 튕겨나가지 않음). 브라우저/앱 전체가 확대되지 않는다.
2. **Ctrl+휠 — 커서 중심 zoom out**
   - 같은 지점에 커서를 두고 `Ctrl`+휠을 **아래로** 굴린다.
   - 기대: 축소되며 커서 아래 지점이 화면상 같은 위치를 유지한다. 최소치에 도달하면 전체가 화면에 맞춰지고(fit) 더 이상 축소되지 않는다.
3. **Shift+휠 — 좌우 이동**
   - `Shift`를 누른 채 휠을 위/아래로 굴린다.
   - 기대: 타임라인이 좌/우로 스크롤된다(세로 이동 아님). 미니맵의 뷰 박스도 함께 이동한다.
4. **수식어 없는 휠 — 세로 스크롤 유지**
   - 아무 키도 누르지 않고 휠을 굴린다.
   - 기대: 트랙 목록이 기존과 동일하게 위/아래로 스크롤된다(zoom·가로 이동이 발생하지 않음).
5. **경계 동작**
   - 최대 확대(TIME_ZOOM_MAX)·최소 축소(fit)에서 각각 계속 굴려도 오류 없이 멈춘다.
   - zoom 후 TIME 슬라이더/미니맵 휠 zoom도 여전히 정상 동작한다.

**[통과 기준]** 위 1~5의 기대 동작이 모두 충족되고, 콘솔 오류가 없다.
