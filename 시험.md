## 2. 시험 항목

### B-197 — 트랙 삭제 후 재생 정지 (v1.14.4)

- **시험 환경 준비**: `npm run build:renderers` 후 `npm start`. 네이티브 엔진까지 검증하려면 `build_native.bat`로 `bin/FocusDAW-AudioEngine.exe`를 먼저 재빌드.
- **시험 항목 / 방법 및 절차**:
  1. 오디오 파일 2개 이상을 로드한다.
  2. **재생 중에** 한 트랙 헤더의 `-` 버튼으로 트랙을 삭제한다 → 해당 트랙 소리가 **즉시 멈추고** 화면에서도 사라지는지 확인(나머지 트랙은 계속 재생).
  3. **정지 상태에서** 트랙을 삭제한 뒤 재생 → 삭제한 트랙은 **소리가 나지 않아야** 함.
  4. 마지막 트랙까지 삭제하면 Project/Playback BPM 표시가 `---`(초기 상태)로 돌아오는지 확인.
  5. (네이티브 빌드 시) 네이티브 엔진 연결 상태에서 2~3을 반복해 동일하게 동작하는지 확인.
- **기대 결과**: 삭제한 트랙이 두 엔진 모두에서 더 이상 들리지 않는다.
