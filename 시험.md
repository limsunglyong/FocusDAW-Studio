## 2. 사전 준비 및 환경 구성

### 2-A. 브라우저 환경 (Fallback 기능 검증)
1. 프로젝트 루트 경로에서 웹 서버를 구동합니다.
   ```bash
   npx http-server -p 8080
   ```
2. Chrome 또는 Edge 브라우저를 열어 `http://localhost:8080/studio.html`에 접속합니다.
3. 브라우저 설정에서 **팝업 차단이 비활성화**되어 있는지 확인합니다.

### 2-B. Electron 앱 환경
1. 프로젝트 루트 경로에서 아래 명령어로 앱을 실행합니다.
   ```bash
   npm start
   ```
