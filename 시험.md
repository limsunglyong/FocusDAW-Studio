# FocusDAW Studio 시험(Test) 계획서

> 대상: **v1.18.8 믹서 Audio In 색 구분 · v1.18.7 크래시 · v1.18.6 반복/볼륨**

> 상태: 수동 확인 대기.

## 1. 시험 항목

### v1.18.8 믹서 Audio In 채널 색 구분 (실동작)

- [ ] 파일 트랙과 Audio In 트랙이 함께 있는 상태에서 믹서 창을 연다.
- [ ] Audio In 채널 스트립이 파일 트랙과 **다른 블루 틴트 배경**으로 표시되는지 확인한다.

### v1.18.7 Audio In 트랙 + 믹서 오픈 시 검은 화면 수정

- [ ] Audio In 트랙을 하나 생성한 뒤(녹음 전) 믹서 창을 연다. **검은 화면/멈춤 없이** 믹서가 정상적으로 뜨는지 확인한다.
- [ ] Audio In 트랙에 녹음을 한 상태에서도 믹서가 정상 표시되는지 확인한다.
- [ ] 파일 트랙만 있는 기존 프로젝트에서 믹서 스펙트럼(FFT) 표시가 이전과 동일한지(회귀 없음) 확인한다.

### v1.18.6-1 Audio In 짧은 트랙 반복 재생 수정

- [ ] 1분 길이 파일 트랙과, 그보다 짧은(예: 1~3초) Audio In 녹음 트랙을 함께 둔 프로젝트를 재생한다.
- [ ] Audio In 트랙이 **1초마다 반복되지 않고**, 녹음된 구간만 재생된 뒤 곡이 끝나 루프할 때 함께 반복되는지 확인한다.
- [ ] 재생 헤드 0이 아닌 위치(예: 5초)에서 녹음한 경우, 재생 시 **그 위치(5초)에서** 소리가 나는지 확인한다.
- [ ] 파일 스템만 있는 기존 프로젝트의 재생/루프가 이전과 동일한지(회귀 없음) 확인한다.

### v1.18.6-2 믹서 창 Audio In 채널 배경색 구분

- [ ] 믹서 창을 열어 Audio In 채널 스트립 배경이 파일 트랙과 **다른 블루 틴트**로 표시되는지 확인한다.
- [ ] 편곡 화면의 Audio In 트랙 헤더 색감과 일관되는지 확인한다.

### v1.18.6-3 마스터 출력 볼륨 조절감 (dB 테이퍼)

- [ ] 믹서 마스터 VOL 페이더를 위→아래로 움직이며 **이동량에 비례해 음량이 고르게 줄어드는지** 확인한다(예전처럼 바닥 근처에서만 급히 작아지지 않아야 함).
- [ ] 페이더의 dB 표시값과 실제 체감 음량이 대체로 일치하는지 확인한다.
- [ ] 채널 볼륨 페이더의 0 dB 마커 위치와 조작감이 자연스러운지, 휠 1 dB 스텝이 정상인지 확인한다.
- [ ] 페이더를 맨 아래로 내리면 완전한 무음(−∞)이 되는지 확인한다.
