1. 소개

FocusDAW는 Stem(드럼, 베이스, 키보드, 보컬 등) 파일들을 불러와 구간 편집, 볼륨 오토메이션, 이펙트 적용 후 MP3/WAV로 믹스다운하는 경량 DAW입니다.

2. 상단 메뉴 & 컨트롤바

Project 드롭다운

  • New Project — 새 프로젝트 시작
  • Open Project — .focus 파일 열기
  • Save Project — 현재 프로젝트 저장
  • Import Stem Folder — 폴더에서 오디오 파일 일괄 불러오기
  • Import Audio Files — 개별 파일 선택
  • Load Demo Session — 데모 4 Stem 로드
  • Export MP3 — 최종 믹스다운

Zoom & Tool

TIME 줌: 슬라이더 또는 +/- 버튼, Minimap 휠

AMP 줌: 파형 진폭 확대/축소

Tool: Select (S), Scissors (C), Join (J)

3. Timeline Minimap 사용법

전체 곡을 한눈에 볼 수 있는 미니맵입니다.

  • 클릭/드래그 → 해당 위치로 빠르게 이동
  • 휠 스크롤 → TIME 줌 In/Out
  • 노란색 브라켓 → 현재 화면에 보이는 영역 표시
  • 세로 플레이바 → 현재 재생 위치

4. 각 Track 기능 상세

Track Header (좌측)

  • Solo (S) — 해당 트랙만 듣기
  • Mute (M) — 해당 트랙 음소거
  • Volume Fader — 트랙 볼륨 조절
  • Pan Knob — 좌우 팬닝
  • VU Meter — 실시간 레벨 표시
  • VOL AUTO — Volume Automation On/Off
  • Trash — 트랙 삭제

Track Lane (타임라인)

  • Scissors (C) — 클릭으로 클립 분할
  • Join (J) — 인접 클립 병합
  • Automation — 파형 위에서 포인트 드래그, 클릭 추가, 우클릭 삭제

5. Mixer 창

Mixer 버튼 클릭 → 드래그 가능한 별도 창이 열립니다.

Master 채널 기능:

  • Graphic EQ · FFT: 9밴드 그래픽 EQ 조절 및 전반적인 스펙트럼 실시간 반응 시각화
  • Level meter: 9대역 주파수별 실시간 LED 레벨 미터 및 EQ 그래픽 오버레이 조절
  • EQ 프리셋 버튼: Flat(Reset), POP, Classic, HIP HOP 프리셋 일괄 적용
  • Reverb / Echo & Delay: 마스터 이펙트 세기 슬라이더

6. Output Effect Track

하단 전용 트랙에서 Master EQ, Fade In/Out을 조절할 수 있습니다.

  • MiniEQGraph: 트랙 헤더에서 현재 적용 중인 9밴드 마스터 EQ 커브의 미니 프리뷰를 실시간 렌더링하여 표시합니다.
  • REV / ECHO 칩: 마스터 Reverb 또는 Echo 이펙트가 켜져 있으면 칩이 하이라이트 활성화 상태가 됩니다.
  • Fade In/Out 핸들: 타임라인 위 삼각 영역의 핸들을 드래그하여 페이드 인/아웃 시간을 정교하게 편집할 수 있습니다.

9. 성능 최적화 및 문제 해결

고사양 데스크톱 환경(예: Intel i9 하이브리드 CPU 등)에서 노트북보다 실행 속도, 피크 로딩, 타임라인 조작, 줌 반응 속도 등이 비정상적으로 느려질 때 조치할 사항입니다.

  • Intel P-core / E-core 스케줄링 최적화: Windows 전원 관리 설정에서 "최고 성능" 모드로 변경하거나 작업 관리자에서 우선순위를 "높음"으로 설정합니다.
  • GPU 하드웨어 가속 점검: 외장 그래픽 카드 환경에서 크로미움 GPU 가속이 비활성화되어 소프트웨어 렌더링으로 줌 속도가 처지는지 드라이버 설정을 점검합니다.
  • NVIDIA G-Sync 및 백그라운드 프레임 제한 해제: NVIDIA 제어판에서 G-Sync 창 모드 버그 및 최대 프레임 속도 제한 해제를 진행합니다.
  • 백신 실시간 감시 예외 설정: 프로젝트 및 오디오 폴더를 백신 실시간 감시 제외 경로로 지정하여 파일 I/O 대기시간을 방지합니다.
